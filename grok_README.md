# Network-Chan Project RAG Documentation

This README serves as the central document for Retrieval-Augmented Generation (RAG) in the Network-Chan project. It provides an overview of key resources and links to dynamic content, allowing the retrieval of the latest information during conversations. For code and files that change frequently (e.g., stubs, modules), refer to the GitHub repository directly for the most up-to-date versions.

## Dynamic Code Repository

To access the latest code (which changes during development), browse the GitHub repo below.

- **GitHub Repo URL**: `https://github.com/SaviorNT/Network-Chan`
  - Example paths to browse:
    - Monorepo structure: /tree/main
    - Appliance code: /tree/main/appliance/src (e.g., telemetry_ingest.py for ingestion logic)
    - Tests: /tree/main/appliance/tests (e.g., test_edge_rl.py for RL coverage)
    - Config/Env: /blob/main/appliance/.env.example (sample env with AUTONOMOUS_MODE)
  - Latest commit: Use /commit/main for changes.
